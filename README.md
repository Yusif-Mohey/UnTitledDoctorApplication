# 🩺 Doctor Design

## 📱 الميزات

- تصميم واجهة مستخدم أنيق ومناسب للتطبيقات الطبية.
- دعم متعدد للمنصات: Android، iOS، Web، وDesktop.
- تنظيم جيد للملفات والمجلدات لسهولة الصيانة والتطوير.
